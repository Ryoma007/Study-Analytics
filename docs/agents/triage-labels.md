# Triage 标签

技能使用五个标准 triage 角色。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签。

| 技能中的标签 | 仓库实际标签 | 含义 |
| ----------- | ----------- | ---- |
| `needs-triage` | `待评估` | 维护者需要评估此 issue |
| `needs-info` | `待补充信息` | 等待提交者补充更多信息 |
| `ready-for-agent` | `可供Agent处理` | 规格完整，AI Agent 可直接接手 |
| `ready-for-human` | `需人工处理` | 需要人类开发者实现 |
| `wontfix` | `不予处理` | 不会采取行动 |

当技能提到某个角色时（例如"打上 AFK-ready triage 标签"），请使用本表右侧对应的标签字符串。

如需调整标签名称，直接修改右侧列即可。
